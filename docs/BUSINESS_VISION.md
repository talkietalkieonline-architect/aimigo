# AIMIGO — Бизнес-концепция и Визия продукта

---

## Дата создания: Июнь 2025 (Сессия 5)
## Источник: интервью с основателем

---

## 1. Суть продукта

Aimigo — AI-first коммуникационная платформа, где пользователи общаются с AI-агентами голосом (и текстом как fallback). Агенты представляют бизнесы, государство, сервисы. В дальнейшем — говорящие аватары с мимикой и одеждой.

---

## 2. Монетизация

### A. Бизнес (B2B)
- **Разовая оплата** — создание агента + его обучение
- **Абонентская плата** — ежемесячная за каждого агента
- **Бюджет на трафик** — генерация голоса, а позже видео
  - Первый этап: заёмные мощности Яндекса
  - Второй этап: собственный «завод» на видеокартах (GPU-ферма)
  - Тарифы на собственных мощностях = тарифы Яндекса (не дороже)

### B. Пользователь (B2C)
- **Подписка** — ежемесячная за:
  - Премиум голоса (расширенная библиотека)
  - Собственный агент-протеже (личный AI-ассистент)
  - Прочие фичи (будет дополняться)

---

## 3. Технологический стек AI

### LLM (языковые модели)
- **Первый этап:** GPT, Claude, Gemini — мультимодельная архитектура
- **Второй этап (с GPU-фермой):** собственная LLM + внешние
- **Стратегия:** разные модели для разной сложности ответов, для текста vs видео

### Голос
- Собственная библиотека голосов (пишем/создаём сами)
- Генерация голоса для агентов

### Визуал / Аватары (будущее)
- Говорящие аватары с мимикой и одеждой
- Первый этап: готовые решения (D-ID / HeyGen / аналоги — на анализе)
- Второй этап: собственная реализация

## 4. Целевые аудитории и сценарии

### D: Кто использует?

| Аудитория | Описание |
|-----------|----------|
| **B2B2C** (основа) | Бизнес создаёт агента → клиенты/пользователи общаются с ним |
| **B2B** | Агент со знаниями для собственного бизнеса (корпоративный) |
| **G2C** | Агент государства для граждан |
| **B2C** | Личный агент-протеже пользователя |

## 5. Режимы платформы (физически разные пространства)

### F: Подробное описание режимов

#### Общение
- Пользователь говорит, узнаёт, поручает, участвует
- **Комнаты:** предустановленные (напр. «Женсовет») + создаёт свои
- Общение с агентами и другими пользователями
- 1-на-1 или групповые комнаты

#### Покупки
- Пользователь попадает в «царство агентов-продавцов»
- Бизнес создаёт внутри платформы **магазин под агента**
- Пользователь: покупает, выбирает, заказывает, откладывает, ждёт скидок
- Агент = продавец-консультант магазина

#### Прогулка
- Агенты-зазывалы привязаны к **геолокации**
- Когда пользователь реально идёт мимо локации — агент приглашает
- Связь физического мира с платформой

#### Обучение
- Образовательные агенты
- Лектории, курсы, тренинги через AI

#### Работа / Бизнес
- Корпоративные агенты
- Внутренние рабочие инструменты через AI

## 6. Эфир (бегущая строка)

- **НЕ комната, а отдельная новостная полоса**
- Приглашённый по рекомендации агент
- Контролирует важную для жизни информацию:
  - Пробки, погода, МЧС, акции, новости платформы
- Работает всегда, на всех экранах

---

## 7. Комнаты

| Тип | Описание |
|-----|----------|
| 1-на-1 | Приватный диалог с агентом или пользователем |
| Групповая | Несколько участников + агенты |
| Предустановленная | Тематические комнаты (Женсовет и т.д.) |
| Пользовательская | Создаётся пользователем для общения с агентами |

## 8. Коммерческий блок (Город Агентов → Бизнес)

### Путь бизнеса:
1. В **Городе Агентов** — кнопка «Для бизнеса» / «Личный кабинет»
2. **Личный кабинет бизнеса:**
   - Профиль компании
   - Статистика агентов
   - Управление магазином (режим Покупки)
   - **Кнопка «Создать агента»** — конструктор
3. **Конструктор агента:**
   - Имя, аватар, голос
   - Промпт / база знаний
   - Сценарии диалогов
   - Привязка к магазину / геолокации / комнате

## 9. UX-решения (Сессия 5)

### Голосовой приоритет
- Платформа voice-first: основной ввод — голос
- Текстовое поле — скрыто по умолчанию, появляется по кнопке ⌨️
- Кнопка Mute убрана — функция привязана к микрофону:
  - Короткое нажатие: вкл/выкл микрофон
  - Длинное нажатие (мик выкл): always-on режим
  - Длинное нажатие (мик вкл): MUTE всех агентов
  - Короткое при MUTE: снять MUTE

### Медиа в голосовом режиме
- Кнопка «+» рядом с микрофоном
- Меню: Фото, Видео, Файл, Ссылка
